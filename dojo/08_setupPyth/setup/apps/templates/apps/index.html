Setting up Django

This page will show you how to set up Django4 from 0 - 100.


This page will also show you how to install Python3.

First of all.... What is Django?

Django is a Python web framework and it is a backend framework, meaning it is on the server side. Django can be used to make your website. Django uses MVT Design Pattern. Hopefully this page can help you get started so you can build out your Django website of your own.

Step 1: Installing Python

  • 1 ) Go to www.python.org
  • 2 ) Go to downloads and select your operating system and download python3 on your computer
  • 3 ) Open your command prompt. Then run "python3 --version" command. If you get the latest version, for example "Python 3.10.4, then means you installed python3 successfully.
